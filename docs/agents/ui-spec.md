## Color Scheme

모노톤 2가지. 시스템 다크모드 따라감.

| Token | Light | Dark |
|-------|-------|------|
| `--bg` | `#FFFFFF` | `#0A0A0A` |
| `--fg` | `#0A0A0A` | `#FFFFFF` |
| `--border` | `#E0E0E0` | `#2A2A2A` |

```css
:root {
  --bg: #FFFFFF;
  --fg: #0A0A0A;
  --border: #E0E0E0;
}
@media (prefers-color-scheme: dark) {
  :root {
    --bg: #0A0A0A;
    --fg: #FFFFFF;
    --border: #2A2A2A;
  }
}
```
